## Foundations & Concepts

- [What Is Business Intelligence](https://www.ibm.com/topics/business-intelligence) – Overview of BI concepts, use cases, and value.
- [BI vs Analytics](https://www.tableau.com/learn/articles/business-intelligence) – Comparison of BI reporting and advanced analytics.
- [Decision Intelligence](https://www.gartner.com/en/information-technology/glossary/decision-intelligence) – Framework connecting BI insights to decisions.
- [Star Schema](https://www.vertabelo.com/blog/star-schema/) – Common dimensional modeling pattern for BI.
- [Kimball Methodology](https://www.kimballgroup.com/data-warehouse-business-intelligence-resources/) – Dimensional data warehousing approach for BI.

## BI Platforms & Tools

- [Tableau](https://www.tableau.com/) – Industry-leading BI platform for interactive analysis.
- [Power BI](https://powerbi.microsoft.com/) – Microsoft BI and reporting platform.
- [Looker](https://cloud.google.com/looker) – BI platform built around a semantic modeling layer.
- [Qlik Sense](https://www.qlik.com/us/products/qlik-sense) – Associative analytics and BI engine.
- [Sisense](https://www.sisense.com/) – BI platform focused on embedded analytics.
- [MicroStrategy](https://www.microstrategy.com/) – Enterprise BI and analytics platform.

## Dashboards & Reporting

- [Looker Studio](https://lookerstudio.google.com/) – Free dashboarding and reporting tool by Google.
- [Tableau Dashboards](https://www.tableau.com/learn/articles/dashboard-design) – Best practices for dashboard creation.
- [Power BI Reports](https://learn.microsoft.com/power-bi/) – Reporting capabilities within Power BI.
- [Mode Analytics](https://mode.com/) – Analytics platform combining SQL, notebooks, and BI.
- [SAP BusinessObjects](https://www.sap.com/products/technology-platform/businessobjects.html) – Enterprise reporting and dashboarding suite.

## Semantic Layers & Metrics

- [LookML](https://cloud.google.com/looker/docs/lookml) – Modeling language for Looker’s semantic layer.
- [dbt Semantic Layer](https://docs.getdbt.com/docs/use-dbt-semantic-layer) – Centralized metrics and definitions for BI.
- [MetricFlow](https://github.com/dbt-labs/metricflow) – Open-source metrics engine.
- [AtScale](https://www.atscale.com/) – Semantic layer and OLAP acceleration for BI tools.
- [Cube](https://cube.dev/) – Headless BI and semantic layer platform.

## Embedded Analytics

- [Sisense Embed](https://www.sisense.com/product/embedded-analytics/) – Embedded BI for applications.
- [Power BI Embedded](https://azure.microsoft.com/products/power-bi-embedded/) – Embedded analytics for custom apps.
- [Tableau Embedded Analytics](https://www.tableau.com/embedded-analytics) – Embedding Tableau dashboards into products.
- [Looker Embedded](https://cloud.google.com/looker/docs/embed-overview) – Embedded analytics using Looker.
- [Apache Superset Embedded](https://superset.apache.org/) – Embedding open-source BI dashboards.

## Self-Service BI

- [Tableau Self-Service](https://www.tableau.com/learn/articles/self-service-analytics) – End-user analytics with governed data.
- [Power BI Self-Service](https://learn.microsoft.com/power-bi/guidance/power-bi-implementation-planning-self-service-bi) – Self-service BI in Microsoft ecosystems.
- [Qlik Self-Service Analytics](https://www.qlik.com/us/solutions/self-service-analytics) – Associative self-service BI.
- [Metabase](https://www.metabase.com/) – User-friendly analytics for non-technical teams.
- [Redash](https://redash.io/) – Query-based BI and dashboards.

## Open-Source BI

- [Apache Superset](https://superset.apache.org/) – Open-source data exploration and BI platform.
- [Metabase](https://www.metabase.com/) – Open-source BI tool for teams.
- [Redash](https://redash.io/) – Open-source SQL-based dashboards.
- [Lightdash](https://www.lightdash.com/) – Open-source BI built on dbt.
- [Helical Insight](https://www.helicalinsight.com/) – Open-source BI and reporting tool.

## Data Modeling & Transformation

- [dbt](https://www.getdbt.com/) – SQL-based transformations for analytics and BI.
- [Power Query](https://learn.microsoft.com/power-query/) – Data preparation and transformation tool.
- [Alteryx](https://www.alteryx.com/) – Data preparation and analytics automation.
- [Matillion](https://www.matillion.com/) – ELT tool for cloud data warehouses.
- [Talend](https://www.talend.com/) – Data integration and preparation platform.

## Visualization Libraries

- [D3.js](https://d3js.org/) – JavaScript library for custom data visualizations.
- [Plotly](https://plotly.com/) – Interactive charts and dashboards.
- [Apache ECharts](https://echarts.apache.org/) – Powerful visualization library for BI dashboards.
- [Vega-Lite](https://vega.github.io/vega-lite/) – Declarative grammar for data visualization.
- [Chart.js](https://www.chartjs.org/) – Simple JavaScript charting library.

## Governance, Security & Access

- [Row-Level Security](https://learn.microsoft.com/power-bi/enterprise/service-admin-rls) – Data access control by user or role.
- [Data Catalogs](https://www.ibm.com/topics/data-catalog) – Discoverability and governance for BI datasets.
- [Collibra](https://www.collibra.com/) – Data governance and catalog platform.
- [Alation](https://www.alation.com/) – Enterprise data catalog for BI and analytics.
- [Okta](https://www.okta.com/) – Identity and access management for BI tools.

## Performance & OLAP

- [SSAS](https://learn.microsoft.com/analysis-services/) – Microsoft OLAP and tabular models.
- [Apache Druid](https://druid.apache.org/) – Real-time OLAP analytics database.
- [ClickHouse](https://clickhouse.com/) – Columnar OLAP database for BI performance.
- [AtScale OLAP](https://www.atscale.com/) – Virtual OLAP for cloud data warehouses.
- [Mondrian](https://community.hitachivantara.com/s/article/mondrian) – Open-source OLAP engine.
